# Technical Architecture Overview

## System Architecture

### Frontend Stack:
- **Next.js 14**: App Router with Server Components
- **TypeScript**: Full type safety
- **Tailwind CSS**: Utility-first styling
- **React Three Fiber**: 3D rendering
- **Mapbox GL JS**: Interactive mapping

### Backend Stack:
- **Supabase**: Primary database and authentication
- **Neon**: Serverless PostgreSQL
- **Vercel**: Deployment and edge functions
- **Blob Storage**: File management

### Key Technical Decisions:

1. **Multi-Database Architecture**: Implemented fallback systems for reliability
2. **Component-Based Design**: Reusable UI components across all pages
3. **Performance-First**: Optimized for mobile and desktop performance
4. **Error Resilience**: Comprehensive error handling and fallbacks
5. **Scalable Structure**: Modular architecture for easy expansion
